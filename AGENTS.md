# AGENTS

- 面向老板与账号运营者的所有界面文案必须使用简体中文，`CPM`、`CPE` 等约定缩写除外。
- 这是同一个 `creator-dashboard` 项目：禁止新建 Cloudflare 项目，禁止改动生产 URL，禁止替换既有上线拓扑。
- 周更只在当前仓库内原位修改：替换 `data/source/latest.xlsx`，重新生成标准化数据，重建现有项目。
- 视觉语言保持克制、高级、低饱和、中性；延续首版的 Bento 卡片、细边框、柔和光晕与大量留白。
- 业务口径必须诚实：不伪造抖音与快手的一一匹配，不把发布时间聚合趋势写成账号日监控。
- 指标公式集中维护在代码配置层，禁止在组件里散落硬编码业务数值或计算公式。
